# College Event Feedback Analysis

## 📊 Overview
This project analyzes student feedback data from a college event to understand satisfaction levels and identify improvement areas.

## 🔧 Tools Used
- Power BI
- Basic Data Analysis

## 📈 Key Metrics
- Ratings
- Feedback categories
- Overall satisfaction

## 🚀 Learning Outcome

- Understood how to visualize insights using dashboards
